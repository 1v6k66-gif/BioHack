# BioHack ЕООД – уебсайт

Статичен маркетингов сайт за BioHack ЕООД — стартъп от Варна за биотехнологични хранителни добавки (сън, хидратация, фокус, тестостерон).

## 🌐 Демо / Структура

Сайтът се състои от 4 HTML страници, споделящи общ хедър, футър и стилове:

| Файл | Описание |
|---|---|
| `index.html` | Начална страница – hero секция, преглед на 4-те продукта, "Защо BioHack", отзиви, CTA |
| `about.html` | За нас – история, мисия/визия/ценности, хронология, екип, предимства |
| `products.html` | Продукти – филтър табове по категория + детайлна секция за всеки продукт (съставки, ползи, дозировка) |
| `contact.html` | Контакти – форма за връзка, информация (адрес, телефон, имейл, работно време), карта-placeholder |

## 📁 Файлова структура

```
/
├── index.html
├── about.html
├── products.html
├── contact.html
├── style.css
├── script.js
└── images/
    ├── logo.png
    ├── knockout.png
    ├── hydro-charge.png
    ├── pure-focus.png
    └── virilion-red.png
```

> ⚠️ HTML файловете сочат към изображенията през `images/logo.png` и т.н. — при качване на сайта създайте папка `images/` и сложете качените PNG файлове (`logo.png`, `knockout.png`, `hydro-charge.png`, `pure-focus.png`, `virilion-red.png`) в нея.

## ✨ Функционалности (`script.js`)

- **Тъмна/светла тема** – превключвател, запазен в `localStorage`
- **Sticky хедър** при скрол
- **Мобилно меню** (hamburger)
- **Бутон "Нагоре"** след скрол
- **AOS анимации** – елементи се появяват при скрол (`IntersectionObserver`)
- **Филтър табове** за продукти по категория (`products.html`)
- **Контактна форма** с валидация и симулирано изпращане
- **Плавен скрол** към anchor линкове (`#knockout`, `#hydro` и т.н.)
- **Брояч анимация** на статистики (напр. "4 продукта", "100%")

## 🎨 Дизайн система (`style.css`)

Цветовете, шрифтовете и разстоянията са дефинирани като CSS променливи в `:root`, с отделен overlay за светла тема (`[data-theme="light"]`):

- **Шрифтове:** `Syne` (заглавия) + `DM Sans` (текст) — Google Fonts
- **Основен акцент:** зелена палитра (`--green-1` до `--green-5`, `#00E6B4`)
- **Тъмна тема по подразбиране**, превключваема на светла

## 🚀 Как да пуснете сайта локално

Тъй като е чисто статичен сайт (HTML/CSS/JS, без build стъпка), просто:

1. Разархивирайте всички файлове в една папка (включително папка `images/` с изображенията).
2. Отворете `index.html` директно в браузър, **или**
3. Стартирайте локален сървър от папката, напр.:
   ```bash
   python3 -m http.server 8000
   ```
   и отворете `http://localhost:8000`.

## 📝 Бележки за доработка

- Контактната форма в момента само симулира изпращане (`setTimeout`) — трябва да се свърже с бекенд/имейл услуга (напр. Formspree, EmailJS или собствен endpoint), за да работи реално.
- Секцията "Карта" в `contact.html` е placeholder — може да се замени с вградена Google Maps карта.
- Всички текстове са на български език (`lang="bg"`).
